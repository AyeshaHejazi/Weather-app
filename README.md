⛅ SkyCast – Weather App
A beautiful, real‑time weather application that fetches live weather data from the OpenWeatherMap API.
Search any city, use your current location, toggle between °C/°F, and keep a list of recent searches – all saved in your browser.

🔗 Live Demo: https://ayeshahejazi.github.io/Weather-app/
Screenshot: Screenshot.png

✨ Features
🔍 Search by city name – worldwide coverage

📍 My Location – uses your device’s geolocation

🌡️ Toggle °C / °F – instant unit conversion

🕒 Recent searches – stored in localStorage (persists after page refresh)

🎨 Dynamic weather icons – based on OpenWeatherMap icon set

⚠️ Error handling – network issues, invalid city, missing API key

📱 Fully responsive – works on mobile, tablet, and desktop

🛠️ Technologies Used
HTML5 – semantic structure

CSS3 – glassmorphism design, flexbox, responsive layout

JavaScript (ES6+) – fetch(), async/await, DOM manipulation, localStorage

OpenWeatherMap API – free weather data
